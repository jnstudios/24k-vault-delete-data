24K Vault - Data Deletion Request

24K Vault primarily stores user data locally on the user's device.

Users can delete their data by:

  • Deleting records within the app.
  • Using any future reset/delete options provided in the app.
  • Uninstalling the app from their device.

If you believe any data associated with our services needs to be removed, please contact:
